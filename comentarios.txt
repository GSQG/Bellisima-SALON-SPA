<nav>:


Se usa para definir el menú de navegación principal de una página web
Contiene los enlaces que permiten navegar por las diferentes secciones del sitio


<section>:


Define una sección independiente de contenido dentro de la página
Agrupa contenido relacionado
Generalmente incluye un encabezado (h1-h6)


<article>:


Representa un contenido independiente y autónomo
Debe tener sentido por sí mismo, incluso fuera del contexto de la página
Por ejemplo: un post de blog, una noticia, un comentario


<header>:


Define la cabecera de una página o sección
Típicamente contiene elementos introductorios o de navegación


<footer>:


Representa el pie de página
Contiene información como derechos de autor, contacto o enlaces relacionados


<p>:


Define un párrafo de texto
Ayuda a organizar el contenido textual en bloques


<form>:


Se usa para crear formularios interactivos
Permite recopilar datos del usuario


<ul> y <li>:


<ul>: Lista desordenada (con viñetas)
<li>: Elementos de la lista
